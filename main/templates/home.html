Create a Superuser

If you need to create a superuser, follow these steps:

1. SSH into your server.

2. Navigate to your Django project directory.

3. Run the following command to create a superuser:

python manage.py createsuperuser

4. Follow the prompts to enter a username, email, and password for the superuser.

Once done, you can access the Django Admin panel at Go to Login and log in using the superuser credentials you created.
